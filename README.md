Automated Harvest Planning Schedule

A Python script utilizing the `datetime` module to calculate estimated harvest dates based on planting dates and Days to Maturity (DTM). This tool aids in succession planting and continuous yield management. 

Key Features

Date Calculation:** Automatically calculates harvest dates by adding DTM to the planting date.
Succession Planning:** Easily schedule staggered plantings for continuous crop availability.
Structured Output:** Provides a clear, columnar schedule summary in the terminal.



 Prerequisites

Python 3.x** installed on your system.



 Usage

1. Save the Code

Save the harvest planning Python code into a file named **`harvest_planner.py`**.

 2. Run the Script

Open your terminal or command prompt, navigate to the directory containing the file, and execute the script:

bash
python harvest_planner.py
